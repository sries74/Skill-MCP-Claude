## Classification Definitions

| Type | Definition |
|------|-----------|
| **template** | Produces concrete, structural output. Has code scaffolding, file patterns, boilerplate, or generation steps that result in files. The output shape is predictable and repeatable. |
| **discipline** | Modifies HOW work is done, not WHAT is produced. Defines processes, constraints, quality gates, or behavioral rules for Claude to follow during generation. |
